## 🎯 About This Portfolio

Every project here follows the same disciplined approach, regardless of dataset:

1. **Understand the problem** before touching code — what's being predicted, what metric matters
2. **Clean data with logic**, not blind statistics, wherever a real-world rule can be applied
3. **Establish a baseline model** before trying anything more complex
4. **Compare model families** rather than committing to the first one that works
5. **Tune systematically** (cross-validated search) instead of hand-guessing parameters
6. **Avoid data leakage** — every statistic used for cleaning/scaling is fit on training data only
